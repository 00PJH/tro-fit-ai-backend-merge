## 🎯 1차 발표 시연 Core Loop (WBS 1단계 기준)

```
1. 앱에서 "측정 시작" 버튼 클릭
2. 카메라로 스쿼트/팔 올리기 동작 수행 (또는 영상 파일 업로드)
3. AI가 관절 분석 → "무릎 ROM 115도, 고관절 85도 — 보통 수준" 결과 표시
4. 부위별 가동성 점수 + 약한 관절 하이라이트 화면 표시
```

> [!TIP]
> 이 4단계 흐름이 완성되면 1차 발표는 성공입니다.  
> 안무 생성 → 피토그래픽은 **2단계 발표에서 시연**합니다.
